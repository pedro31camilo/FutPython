# FutPython

Repositório de estudos em Python focado em visualização de dados e análise exploratória, com exemplos aplicados ao contexto do futebol.

## Objetivo

Este projeto reúne notebooks e dados utilizados para explorar diferentes técnicas de visualização, incluindo:

- gráficos de dispersão e distribuição
- visualizações de posicionamento e passes
- mapas de chutes e eventos
- gráficos radar e comparativos
- clustering e análise de padrões
- modelos de expectativa de gols (xG)

## Estrutura do repositório

- notebooks com exemplos práticos de visualização e análise
- dados em CSV para reproduzir os estudos
- materiais auxiliares em pastas específicas para download e experimentação

## Como usar

1. Clone o repositório em sua máquina.
2. Crie um ambiente virtual com Python.
3. Instale as dependências necessárias para os notebooks.
4. Abra os arquivos .ipynb no Jupyter Notebook ou JupyterLab.

## Tecnologias e bibliotecas

Os estudos podem envolver bibliotecas como:

- Python
- Jupyter Notebook / JupyterLab
- pandas
- matplotlib
- seaborn
- numpy
- scikit-learn

> As dependências podem variar conforme o notebook, então é recomendável revisar cada arquivo antes de executá-lo.

## Observação

Este repositório tem caráter acadêmico e de estudo, com foco em aprender e experimentar técnicas de visualização de dados com Python.
